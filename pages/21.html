NPM

  • NPM работает в среде node.js
  • Проверить версию node.js:
    •   
    node -v
  • Сайт NPM
  • Проверить версию NPM:
    •   
    npm -v
  • Для инициализации NPM в проекте:
    •   
    npm init
  • Для пропуска 'вопросов' и создания дефолтного package.json:
    •   
    npm init -y
  • Запуск скриптов из package.json
    •   
    npm run (название скрипта из раздела 'scripts')
  • Установить пакеты (пример библиотека lodash)
    •   
    npm i lodash
  • Чтобы не пересылать зависимости в удалённый репозиторий, в корне проекта создаётся фаил .gitignore
    •   
    //в самом файле указывается:
    /node_modules
  • dev зависимости (пакеты, необходимые только во время разработки). Если не указать -D, зависимость установится как основная
    •   
    //для примера установить eslint
    npm install eslint -D
  • Удаление пакетов:
    •   
    npm uninstall lodash
  • Удалять node модули (папку целиком) нужно рекурсивно с помощью команды:
    •   
    Remove-Item -Path node_modules -Recurse -Force
