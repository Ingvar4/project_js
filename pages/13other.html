Mock

Mock - это объект, который имитирует поведение реального объекта или системы в тестировании.

Ссылки на Mock - сервисы:

  1. JSONPlaceholder

    Бесплатный онлайн-сервис, который предоставляет фейковый REST API с JSON-ответами. Подходит для тестирования и прототипирования. ссылка

  2. MockAPI

    Сервис для создания mock API с поддержкой CRUD операций. Позволяет генерировать JSON-данные и настраивать структуру API. ссылка

  3. Beeceptor

    Инструмент для создания mock API с возможностью перехвата запросов и настройки JSON-ответов. ссылка

  4. Mockaroo

    Mockaroo позволяет генерировать большие объемы реалистичных тестовых данных в формате JSON. Вы можете настроить структуру данных и скачать JSON-файл или использовать API для получения данных. ссылка

  5. Mocky

    Mocky позволяет создавать кастомные JSON-ответы и предоставляет уникальную ссылку для доступа к ним. Подходит для быстрого создания mock-заглушек. ссылка

  6. ReqRes

    Бесплатный сервис для создания mock API. Позволяет отправлять запросы и получать JSON-ответы. ссылка

  7. FakeJSON

    Сервис для генерации JSON-данных на основе вашего шаблона. Поддерживает кастомизацию и предоставляет API для доступа к данным. ссылка

  8. Postman Mock Server

    Postman позволяет создавать mock-серверы с JSON-ответами. Требуется регистрация и настройка в Postman. ссылка

  9. JSONBin

    Сервис для хранения и управления JSON-данными. Позволяет создавать и использовать JSON-заглушки. ссылка

  10. Npoint

    Простой инструмент для создания и хранения JSON-данных с уникальной ссылкой для доступа. ссылка
