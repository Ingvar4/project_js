Dom.

Навешивание слушателя событий на элемент с идентификатором 'myId' (к примеру).

  
    document.getElementById('myId').addEventListener('click', () => {
      console.log('Привет');
    });

Пример, если нужно навесить слушатель событий на все элементы коллекции с классом '.myClass' (к примеру).

  
    document.querySelectorAll('.myClass').forEach((el) => {
      el.addEventListener('click', () => {
        console.log('Hello');
      });
    });
