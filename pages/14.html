Регулярные выражения.

Регулярные выражения это шаблоны, которые используются для сопоставления последовотельностей символов в строках.

Синтаксис:

  
    regexp = new RegExp("шаблон", "флаги");
    //Объявление регулярного выражения через new RegExp
    let regexp1 = new RegExp('^[а-яА-Я]*$'); //без флагов
    let regexpFlag1 = new RegExp('^[а-яА-Я]*$', 'gmi'); //с флагами gmi
    //или 
    regexp = /шаблон/; // без флагов
    regexp = /шаблон/gmi; // с флагами gmi
    //Объявление регулярного выражения c использованием слешей:
    let regexp2 = /^[а-яА-Я]*$/; //без флагов
    let regexpFlag2 = /^[а-яА-Я]*$/gmi; //с флагами gmi
